performance.now() 获取当前时间

在新的 requestAnimationFrame 规范中，frame 回调的参数 timestamp 是一个 DOMHighResTimeStamp 对象，它比 Date 的计时要更精确（可以精确到纳秒）。因此获取时间我们优先使用 performance.now()，如果浏览器不支持 performance.now()，我们再降级使用 Date.now()。
