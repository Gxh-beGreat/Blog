动画的简易封装

为了实现更加复杂的动画，我们可以将动画进行简易的封装，要进行封装，我们先要抽象出动画相关的要素：

  • 动画时长：\(T = duration\)
  • 动画进程：\( p = \frac{t}{T}\ (p \in [0, 1])\)
  • 动画量子 easing：\(e = f(p)\)
  • 动画方程：\([x, y] = G(e) = G(f(p))\)
  • 动画生命周期：开始、进行中、结束。

start 方法包含一个参数，这个参数是一个布尔类型或者回调函数，当动画结束的时候，如果这个参数是回调函数，将执行这个函数，它的返回值如果不是 false 那么结束动画，否则循环播放动画。
